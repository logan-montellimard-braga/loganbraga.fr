Logan Braga Personal Website
============================

[Website](http://loganbraga.fr).

Version 3.

## Stack

+ AngularJS
    - Angular Route
    - Angular Animate
+ lodash
+ Modernizr
+ FastClick
+ ParticlesJS
+ Google Maps API
+ Google Fonts API
+ Foundation
+ Normalize
+ FontAwesome

For back-end stack, see [api.loganbraga.fr](https://github.com/loganbraga/api.loganbraga.fr).
